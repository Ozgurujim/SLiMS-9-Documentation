### System environment setting

------

This menu item allows the setting of the system environment mode.

* **System Environment mode** [Production/Development] (default = Production). *Choosing Development will enable debug message details to be displayed if errors occur , and this mode should always be enabled when troubleshooting.*
* **Environment for some IP?** [Enable/Disable] (default = Disable). *If enabled, allows control of which IP addresses on which clients  will see the debug messages.* 
* **IP range ...** .*The range of  IP addresses that will be impacted if the above setting is enabled. If enabled, the range would usually be for fixed IP addresses of system administrator and/or librarian. Left empty, it will impact all addresses*
