The Master File menu is used to manage look-up files  that act as authority files for cataloguing , such as GMD, publisher, author, subject, and so on. Master file settings are useful for simplifying the library manager's work in processing bibliographic data, and ensuring consistency and data integrity. 

A valuable explanation on the role of authority files in cataloguing and optimising user searching can be found at https://www.scisdata.com/connections/issue-99/the-lowdown-on-authorities-and-authority-files/

------

In this module we can enter the master file data that can be used in bibliographic data entry, and also manage it. The data that is defined in this Master File module is:

- GMD General Material Designation

- Content type

- Media type

- Carrier type

- Publisher

- Supplier

- Author

- Subject

- Location

- Place

- Collection type

- Doc. language

- Label

- Frequency

  

  ###### The following tools are also provided to help manage some authority files, and other settings.

  

- Visitor room

- Comment management

- Cataloguing Servers

- Item code pattern

- Orphaned author

- Orphaned subject

- Orphaned publisher

- Orphaned place
