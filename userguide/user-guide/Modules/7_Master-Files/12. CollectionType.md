#### This sub-menu is used to manage the Collection Type lookup file .

Resources owned by the library, can be grouped by type, for example: Textbook, Reference, Fiction.

<u>These collection types  are used when setting Loan rules in the Circulation module</u> e.g An item of type = Reference might be restricted to only one day loan, or to a special group of library members. 

**Collection type list**

This menu enables management of the collections type master-file. It displays  the list of all collection types in the SLiMS database , with data for:

- *Collection type* ( name of the collection category)
- *Last update* ( When record was last edited)

This section is provided with facilities to DELETE  and EDIT collection data.

To edit an item , double-click on the collection type , or single-click on the pencil (edit) icon.

A search function allows you to search for entries by keywords.

Results can be sorted by clicking on the field name at the top of each column. 

##### Add new collection type

This provides the facility to add new collection types into the SLiMS system. 

##### Delete collection type

A collection type must be selected first, and after clicking the DELETE SELECTED DATA button a requester  will appear, asking for confirmation.

If the collection type is in use in any existing catalogue records, it cannot be deleted, and a notification will  appear.
