#### This sub-menu is used to manage the Visitor rooms .

Lists and manages the enabled Visitor rooms.

##### Room list

Displays the list of rooms, with data for:

- *Room name*

- *Room code*

  


This section is provided with facilities to DELETE  and EDIT Visitor rooms.

To edit an item , double-click on the room, or single-click on the pencil (edit) icon.

A search function allows you to search for entries .

Results can be sorted by clicking on the field name at the top of each column. 

**Add new room**

This provides the facility to add new visitor rooms directly to the  data in the SLiMS system. Visitor Room information includes the fields  listed above, [ the room code will be randomly generated ] with the exception of *Last updated*, which is done automatically when the **Save** button is clicked.

**Note:** *As of SLiMS 9.7.2 the Visitor room has no clear function.*
